## Un clic. Le profil dans Join.

Naviguez sur LinkedIn comme d'habitude. Cliquez sur l'icône Join depuis n'importe quel profil : la personne arrive là où vous en avez besoin, dans le pipeline d'un poste précis ou dans le vivier pour plus tard. Nom, poste, localisation, parcours et coordonnées publiques sont récupérés automatiquement depuis le profil.

- Naviguez sur LinkedIn comme d'habitude, puis cliquez sur l'icône Join pour enregistrer
- Nom, poste, localisation et parcours récupérés automatiquement
- Envoyez le profil directement dans le pipeline d'un poste ou dans le vivier
