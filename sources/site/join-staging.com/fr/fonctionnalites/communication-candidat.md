## Une seule boîte. Les réponses là où vous travaillez.

Chaque conversation tient dans un seul fil, sur le profil du·de la candidat·e. Quand la personne répond, Join transfère sa réponse vers votre e-mail professionnel : vous répondez directement de là. La réponse se synchronise automatiquement sur le profil, pour que l'équipe garde toujours le contexte.

- Chaque conversation dans un seul fil sur le profil du·de la candidat·e
- Les réponses des candidat·es transférées automatiquement vers votre e-mail professionnel
- Les réponses se synchronisent sur le profil : l'équipe garde toujours le contexte

## Des modèles qui sonnent vraiment personnels.

Rédigez à l'avance l'invitation au tour suivant, le refus, la relance après une offre. Les champs de fusion comme {first\_name}, {role} et {interview\_date} reprennent les données du profil : chaque envoi se lit comme un message individuel. Enregistrez les modèles que votre équipe utilise le plus, et ajustez-les directement avant l'envoi.

- Les champs de fusion reprennent le nom, le poste et les dates depuis le profil
- Enregistrez les modèles que votre équipe envoie le plus souvent
- Ajustez le texte avant l'envoi, aucun modèle figé

## Programmez l'e-mail pour le meilleur moment.

Vous rédigez le refus à 23 h ? Programmez son envoi pour mardi à 10 h, quand la personne le lira plus volontiers sans que cela gâche sa soirée. Idem pour l'invitation au tour suivant. La file d'envoi est visible pour chaque candidat·e : annulez ou reprogrammez à tout moment avant le départ de l'e-mail.

- Programmez vos envois plusieurs jours à l'avance, à l'heure près
- Annulez ou reprogrammez à tout moment avant la livraison
- File d'envoi visible pour chaque candidat·e, aucun angle mort
