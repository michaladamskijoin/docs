Puis-je héberger la page carrière sur mon propre domaine ?

Deux voies, choisissez celle qui vous convient. Voie A : Join héberge la page carrière sur votre sous-domaine Join (votreentreprise.join.com/careers), prête en quelques minutes avec SSL gratuit. Voie B : intégrez le widget d'offres Join dans votre site d'entreprise existant (sur WordPress, Webflow, Wix ou n'importe quelle page HTML), de sorte que les offres s'affichent sur votre propre domaine, avec votre force SEO existante. Nous ne proposons pas de troisième option (une page hébergée par Join sur votre domaine personnalisé) : la voie d'intégration couvre le même besoin avec moins de tracas DNS et un meilleur SEO.

Faut-il un·e développeur·euse pour la mise en place ?

Non. L'éditeur de page carrière est sans code. Téléchargez votre logo, définissez votre palette de couleurs, publiez. Un·e développeur·euse n'est utile que si vous souhaitez intégrer le widget d'offres Join à votre site existant (sur WordPress, Webflow, Wix ou en HTML simple).

Ma page carrière apparaîtra-t-elle sur Google for Jobs ?

Oui. Chaque page carrière est livrée avec un balisage JobPosting structuré qui rend vos offres éligibles à Google for Jobs, plus une sitemap qui soumet les nouvelles offres automatiquement aux moteurs de recherche.

La page carrière fonctionne-t-elle dans d'autres langues que le français ?

Oui. La page carrière s'affiche automatiquement dans la langue du navigateur du·de la candidat·e. Une personne qui navigue en français voit l'interface en français, en allemand elle la voit en allemand. Les offres elles-mêmes s'affichent dans la langue dans laquelle elles ont été rédigées.

Puis-je intégrer les offres Join dans mon site existant ?

Oui. Insérez le widget d'offres Join dans WordPress, Webflow, Wix ou n'importe quel site HTML. Les offres se mettent à jour en direct depuis votre workspace Join ; l'habillage reste le vôtre.
