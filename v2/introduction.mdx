The Example API is a REST API for creating and managing items. It uses
predictable resource-oriented URLs, returns JSON, and authenticates with bearer
tokens.

<Note>
  You are viewing the **v2** docs — the latest version. New in v2: [webhooks](/v2/guides/webhooks),
  [cursor pagination](/v2/guides/pagination), item timestamps, and partial
  updates (`PATCH`). Coming from v1? See the [API reference overview](/v2/api-reference/overview).
</Note>
