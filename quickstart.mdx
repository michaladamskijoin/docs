---
title: "Quickstart"
description: "Run the docs locally and make your first edit."
---

## Install the CLI

```bash
npm i -g mint
```

## Run locally

From the folder that contains `docs.json`:

```bash
mint dev
```

Your preview is available at `http://localhost:3000`.

## Make an edit

<Steps>
  <Step title="Edit a page">
    Open any `.mdx` file and change its content. The preview reloads automatically.
  </Step>
  <Step title="Update navigation">
    Add or reorder pages in the `navigation` section of `docs.json`.
  </Step>
  <Step title="Publish">
    Push to your default branch to deploy the changes.
  </Step>
</Steps>
