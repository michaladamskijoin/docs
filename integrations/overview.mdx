Join connects to the tools your hiring team already uses — from your calendar and video conferencing apps to your HRIS, Slack, and beyond. Whether you want automatic interview scheduling, real-time Slack notifications on new applications, or a fully custom integration through the API, Join gives you the building blocks to run a faster, more connected hiring process.

## Which integrations require a paid plan?

Some integrations are available on every plan, while others require Advanced or Enterprise. Use the table below to check what's included on your plan.

| Integration | Standard | Advanced | Enterprise |
|---|:---:|:---:|:---:|
| Calendar sync | ✓ | ✓ | ✓ |
| Chrome extension | ✓ | ✓ | ✓ |
| Jobs widget embed | ✓ | ✓ | ✓ |
| Zapier & Make | ✗ | ✓ | ✓ |
| REST API | ✗ | ✓ | ✓ |
| Candidate export | ✗ | ✓ | ✓ |
| Custom integrations | ✗ | ✗ | ✓ |

## Jobs widget

Embed your open jobs directly on your company website using the Join jobs widget. The widget works with any platform — WordPress, Webflow, Wix, or plain HTML — and stays in sync with your live job listings automatically. See [Careers Page Setup](/setup/careers-page) for embed instructions.
