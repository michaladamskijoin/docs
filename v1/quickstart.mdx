## 1. Get an API key

Create a key in your [dashboard](https://dashboard.example.com) under
**Settings → API keys**. See [Authentication](/v1/authentication) for details.

## 2. List items

The v1 list endpoint returns the full collection as a JSON array.

<CodeGroup>

```bash cURL
curl https://api.example.com/v1/items \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const res = await fetch("https://api.example.com/v1/items", {
  headers: { Authorization: "Bearer YOUR_API_KEY" },
});
const items = await res.json();
```

```python Python
import requests

res = requests.get(
    "https://api.example.com/v1/items",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
)
items = res.json()
```

</CodeGroup>

## 3. Create an item

```bash
curl -X POST https://api.example.com/v1/items \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "My first item"}'
```

A successful request returns the created item with its generated `id`.
