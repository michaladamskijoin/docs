The Example API is a REST API for creating and managing items. It uses
predictable resource-oriented URLs, returns JSON, and authenticates with bearer
tokens.

<Note>
  You are viewing the **v1** docs. v1 is stable and supported. New projects
  should use [v2](/v2/introduction), which adds webhooks, pagination, and
  partial updates.
</Note>
